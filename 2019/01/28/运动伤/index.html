雨天

发表于 2019-01-28 | 分类于 杂记


上周六去参加飞盘运动，为了接住一个盘，起跳后失去重心，手被身体压到，并折到了手，非常的疼痛，并在接下来的一周里出现红肿、淤青等情况，而且手用不上力气，痛苦不堪。
前天，也就是周六，照常早上起床完成早上的书写任务，然后拿起手机准备回复昨天看到的飞盘活动的情况。我表达了手伤未愈，想修养的意思。有几个盘友半开玩笑似的说没事，单手也可以玩，暂时没被说动。因为小蛇说了中午会来我这一趟，我便建议我做饭，到了一起吃午饭。之后看了会书，起床洗漱后就开始准备做饭材料，两道菜——分别茄子和四季豆炒肉。因为距离上次受伤已经过了一周，即使没用任何药物，但现在也基本消肿了，即使用不上大力气，也还是可以做做饭等这些的。前后准备、炒菜，直到饭菜全部完成，差不多将近一个小时了，这也是好多人不想自己做饭的原因之一吧。做饭期间君君发消息让我去飞盘，而后小蛇也建议下午一个人在家太无聊，也建议我一起去，意志不够坚定的我在心里做了去的决定。
最终我和小蛇一起到了应技大（玩飞盘的地方）。飞盘真的是项神奇的运动，充满了静和动两面，之后心静下来，不那么慌张着急以后，才可以传出优美的盘，或是圆润的弧线，亦或是不带任何偏转的直线。而接盘的时候则需要全身动起来，眼睛、手臂、腿脚的完美配合才可以让你接住一个飞来的盘，当接住一个有难度的盘后的心情充满了难以言表的兴奋。人到了差不多后，便组织了比赛，我作为一个半残人士加入。原本想着跑的慢一些，不要勉强自己，但一上了场，所有这些就都抛在了脑后，带着原始的冲动去追每个盘，好像飞行中的盘对我有种深深的魔力，吸引我，引导着我释放自己。当我在追逐一个盘的时候，我处在了完全的个人世界，周遭的一切都与我无关，天地间只剩下我和盘了，而我的目标就是追上它，摘下它，然后重回现实的世界，这种忘却世界的情况在其他活动体验中绝无仅有。我的传盘也因为每个工作晴好天气的中午和同事一起的练习，有了些许的进步，目前最大的问题还是对于场上形势的判断，老是会做出很多很差的预判。
今天运动的前半段时间，因为抢盘发生了一次冲撞，但并未引起任何伤痛，只是衣服脏了些。但是后半段出现了始料不及的事情，又是同样跳起来接盘，之后稍稍失去重心，居然用了我的伤手着了地，虽短短的一瞬间，但那种深入骨髓的疼让我表情异常的狰狞，钻心的疼让我对任何事物失去了关注。场下坐了缓了很久，好像好了些又重新上场奔跑。期间因为失误还误伤了jumping，心里充满内疚。
通过以上的事情可以总结一下，在自己受伤的情况下不要来参加比赛，也别带着我跑慢点这样的想法，因为一到场上我早就把这些想法抛到脑后了，完全受控于我的本能，其次身体因伤痛带来的不协调，往往还会导致动作变形、发力方式不对等情况，甚至会误伤自己的伙伴。所以我决定在我完全修养好之前，不在参加比赛活动，避免伤人伤己。谨记！
